## 기술 스택 (Technique Used)
### Server(back-end)
- Python : 3.8.12 
- Django : 3.2.9 
- Django REST framework : 3.12.4 
- MariaDB : 10.6.4
- minIORedis : 6.2.6


### Front-end
- Flutter : 2.5.3
- Dart : 2.14.4
- getX : 4.3.8



### Infra
- Ubuntu-Server : 20.04.3 LTS 
- Docker
- Nginx
- Git & GitHub
- GitHub Actions

### Cooperation
- Dooray
- Zoom
